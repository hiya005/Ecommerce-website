# Opal Vogue – Front-End E-Commerce Website

## Project Overview
Opal Vogue is a front-end e-commerce website developed using HTML5 and CSS3.
The project represents an online fashion store interface that showcases
products, categories, offers, and customer-focused sections with a clean
and responsive design.

This is a static website project with no back-end or database integration.

---

## Objectives
- Design a modern and responsive e-commerce website layout
- Practice semantic HTML structure
- Apply CSS for styling, layout, and responsiveness
- Understand real-world front-end UI design concepts

---

## Technologies Used
- HTML5 – Website structure
- CSS3 – Styling and responsive design
- Google Fonts – Typography
- Visual Studio Code – Development environment

## Features
- Header with search and user action buttons
- Hero section with promotional banner
- Flash sale display
- Product categories and product cards
- Sidebar filter section (UI only)
- Best sellers section
- Customer testimonials
- Newsletter subscription form
- Responsive layout for different screen sizes

---

## Conclusion
Opal Vogue demonstrates the implementation of a complete front-end
e-commerce website using HTML and CSS. The project strengthened
understanding of layout design, responsive development, and UI styling.
